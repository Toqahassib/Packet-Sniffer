# Packet-Sniffer


This multi-user packet sniffing tool is listens to traffic originating from clients and displays them on serverâ€™s console. For example, the client sends data packet to the server, and the server extracts the clients MAC address, IP address, and port address from the received packet. Then the server displays extracted information on the console. The tool also gives the user the ability to select which protocol (TCP or UDP) to monitor.


PRE-REQUISITES:
- Unix Operating System.
- Installing Scapy, install scapy in your operating system.
